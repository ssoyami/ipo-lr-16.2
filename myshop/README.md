# Магазин (Django проект)

Это простой проект магазина, выполненный на Django.

## Страницы

- Об авторе - информация о создателе проекта: /about/
- О магазине - информация о теме работы: /shop-info/
- Главная страница - с ссылками на первые две: /main/

Цель проекта — продемонстрировать создание базового сайта на Django с множеством страниц и навигацией.
